Primer：給非眼科醫師的背景補充

Tip

眼科醫師可以跳過這章。 這是給跨科背景（內科、血液腫瘤、家醫、研究員）或非醫師助教的補習。

這份 primer 不是教科書節錄，是「進這堂課前 30 分鐘該知道的所有東西」。 讀完你應該能跟眼科醫師同桌討論 paper、看 OCT 不會迷路、知道 faricimab 為什麼是大事。

1. 我們今天的舞台：nAMD（neovascular age-related macular degeneration）

屬性 nAMD
中文 新生血管性年齡相關黃斑部退化
患者 50 歲以上，全球 ~2,000 萬人、台灣 ~30 萬
病理 黃斑部底下長異常新血管 → 漏液 / 出血 → 視網膜結構破壞 → 視力急速喪失
失明速度 不治療：3 個月失中央視力；治療：可穩定甚至改善
治療年代 2006 anti-VEGF 出現後完全改寫

為什麼是大事？ - nAMD 是西方國家老年失明 #1 原因 - 沒治療 → 失明、無法駕車、生活自主性崩盤 - anti-VEGF 是現代藥物史最成功的眼科例子（救回的視力數量級可比 statin 救回的心血管事件）

2. 眼底 30 秒解剖（你需要的部分）

關鍵詞： - macula（黃斑部）：視網膜中央 ~5 mm 區，負責中央銳利視力（讀字、看臉） - fovea（中央凹）：黃斑部正中央 1 mm，這裡的視覺細胞密度最高 - RPE（retinal pigment epithelium，色素上皮）：視網膜底下一層黑色色素細胞，nAMD 的新生血管是從這層底下竄上來的 - choroid（脈絡膜）：RPE 後面的血管層，是新生血管的源頭

心臟科類比：黃斑部之於視網膜，像左前降支 LAD 之於冠狀動脈——出問題後果最嚴重。

3. nAMD 的 pathophysiology（30 秒版）

治療目標：把 VEGF（及 Ang-2）關掉 → 新血管萎縮 → 漏液停止 → 視網膜恢復。

4. anti-VEGF：眼科最大的藥物突破

VEGF-A（vascular endothelial growth factor A）= 血管生成的核心訊號。 anti-VEGF 藥 = 直接綁住 VEGF-A 不讓它作用。

怎麼給藥： - 玻璃體內注射（intravitreal injection, IVI）——細針從眼白旁邊插進玻璃體，直接把藥推到眼球內 - 局部痛感、無系統毒性 - 每 4–16 週一次，依藥 / 病情而異

藥 廠商 機制 上市
ranibizumab (Lucentis) Genentech/Roche/Novartis anti-VEGF-A 2006
aflibercept 2 mg (Eylea) Regeneron/Bayer anti-VEGF-A + VEGF-B + PlGF 2011
bevacizumab (Avastin, off-label) Roche anti-VEGF-A（原為腫瘤藥） 仿單外
brolucizumab (Beovu) Novartis anti-VEGF-A 小分子 2019（後因 vasculitis 副作用受限）
aflibercept 8 mg (Eylea HD) Regeneron/Bayer 高劑量版 2023
faricimab (Vabysmo) Roche/Genentech anti-VEGF-A + anti-Ang-2 雙標靶 2022

5. faricimab 為什麼特別

faricimab 是全球第一個眼科雙特異性抗體（bispecific antibody），同時綁：

  1. VEGF-A——標準 anti-VEGF 路徑
  2. Ang-2（angiopoietin-2）——血管不穩定的訊號

藥理意義： - aflibercept 把 VEGF 擋住但 Ang-2 還在，血管「不穩」這一塊沒解 - faricimab 同時擋兩條路徑 → 血管更穩 → 漏液更快好 → 可以拉長注射間隔（最長 Q16W vs aflibercept 的 Q8W）

對 paper 的意義：今天讀的 Cheung 2025 就是在比較「faricimab 的 anatomic outcome（CST、SRF）是不是比 aflibercept 好」——這直接驗證雙標靶 hypothesis。

6. OCT 一張圖看懂

OCT（optical coherence tomography）：用紅外光做的「視網膜超音波」。非侵入性、20 秒內掃完一隻眼睛、空間解析度 ~5 μm。

OCT 切片看起來像這樣（橫切視網膜）：

正常 CST ≈ 250–280 μm。nAMD 患者 ≈ 300–500+ μm（變厚是因為水腫）。

OCT 上看 fluid 兩種

類別 位置 意義 治療 priority
IRF (intraretinal fluid) 視網膜各層內的水泡 比較不好，提示主動 disease 高，必須積極治療
SRF (subretinal fluid) 視網膜下方 vs RPE 之間 比較不確定，少量 SRF 可能保護視網膜 中
PED (pigment epithelial detachment) RPE 跟 Bruch 膜分離 nAMD 標誌之一 看大小

治療目標：把 IRF + SRF 都消掉（“fluid-free retina”）。 Cheung 2025 的核心 outcome：「faricimab 比 aflibercept 多少病人在 week 12 fluid 全消？」

7. 視力怎麼量

Snellen vs decimal vs ETDRS letters

系統 例子 用在哪
Snellen 6/6, 20/20, 6/12, 20/40 臨床日常
decimal 1.0, 0.5, 0.1 台灣 / 日本臨床
ETDRS letters 0–100 letters 眼科 trial 唯一標準

換算速記：

ETDRS letters Snellen decimal
100 20/10 2.0
85 20/20 (6/6) 1.0
70 20/40 (6/12) 0.5
55 20/100 (6/30) 0.2
35 20/200 (6/60) 0.1（法定低視能）
0 20/2000 0.01

為什麼用 letters？ - 連續變項（適合 t-test、MMRM） - 每行 5 letter，1 行 = 5 letter（讀懂這條就夠用） - 「進步 15 letters」≈ 「視力進步 3 行」≈ 從 20/40 到 20/20，是臨床顯著的一級事件

8. 給藥節奏：Q4W / Q8W / T&E

「Q」 = 每，後面數字 = 週。 - Q4W = 每 4 週一針（loading dose 期通常這樣） - Q8W = 每 8 週一針（aflibercept 標準） - Q12W / Q16W = 每 12 / 16 週一針（faricimab 可拉到的）

Treatment-and-extend (T&E)

「依病情拉長間隔」的策略： 1. 先 Q4W × 3 針（loading） 2. 病情穩定 → Q8W 3. 再穩 → Q12W 4. 再穩 → Q16W 5. 一旦復發 → 縮回去

為什麼重要？ 打針是病人的痛點：每次都要請假、家屬接送、心理壓力。減少打針次數 = 大幅改善生活品質與健保成本。 faricimab 的 ~80% 患者 1 年後可以 Q12W 以上 → 比 aflibercept 顯著減少打針負擔。

9. anatomic 與 functional outcome 的「不平行」

很多眼科 trial 會看到：

兩個藥的 CST / fluid（anatomic）有顯著差異，但 BCVA（functional）差不多。

Cheung 2025 也是這樣。為什麼？

  • CST / fluid 是 disease activity 的客觀 readout → 對藥很敏感
  • BCVA 是 final functional output，受很多因素影響：
    • Photoreceptor（感光細胞）已死的不會回來
    • Outer retina structure 已破壞的恢復有限
    • 視神經狀況、白內障、其他眼疾
  • 短期（12 週）：解 fluid 快不一定立刻反映在 BCVA
  • 長期（48 週+）：anatomic control 好的人比較不容易視力惡化

對你讀 paper 的意義： 看到「anatomic 改善但 vision 沒改善」不要急著說藥沒用——這是 nAMD 領域常態。

10. 進階：Ang-Tie2 路徑

如果你想了解為什麼 faricimab 加 Ang-2 是有道理的：

faricimab 同時： - 擋 VEGF-A（不讓新血管長出來） - 綁 Ang-2（讓 Tie2 能恢復、血管穩定）

→ 解 disease 的兩個面向，一次處理。

|---|---|---|---|
| **IRF** (intraretinal fluid) | 視網膜各層**內**的水泡 | 比較不好，提示主動 disease | 高，必須積極治療 |

**Cheung 2025 的核心 outcome**：「faricimab 比 aflibercept 多少病人在 week 12 fluid 全消？」

## 7. 視力怎麼量

### Snellen vs decimal vs ETDRS letters

| 系統 | 例子 | 用在哪 |
|---|---|---|
| **Snellen** | 6/6, 20/20, 6/12, 20/40 | 臨床日常 |
| **decimal** | 1.0, 0.5, 0.1 | 台灣 / 日本臨床 |
| **ETDRS letters** | 0–100 letters | **眼科 trial 唯一標準** |

**換算速記**：

| ETDRS letters | Snellen | decimal |
|---|---|---|
| 100 | 20/10 | 2.0 |
| 85 | 20/20 (6/6) | 1.0 |
| 70 | 20/40 (6/12) | 0.5 |
| 55 | 20/100 (6/30) | 0.2 |
| 35 | 20/200 (6/60) | 0.1（**法定低視能**）|
| 0 | 20/2000 | 0.01 |

**為什麼用 letters？**
- 連續變項（適合 t-test、MMRM）
- 每行 5 letter，**1 行 = 5 letter**（讀懂這條就夠用）
- 「進步 15 letters」≈ 「視力進步 3 行」≈ 從 20/40 到 20/20，是臨床顯著的一級事件

**ETDRS chart**：

## 8. 給藥節奏：Q4W / Q8W / T&E

「Q」 = **每**，後面數字 = 週。
- **Q4W** = 每 4 週一針（loading dose 期通常這樣）
- **Q8W** = 每 8 週一針（aflibercept 標準）
- **Q12W / Q16W** = 每 12 / 16 週一針（faricimab 可拉到的）

### Treatment-and-extend (T&E)

「依病情拉長間隔」的策略：
1. 先 Q4W × 3 針（loading）
2. 病情穩定 → Q8W
3. 再穩 → Q12W
4. 再穩 → Q16W
5. 一旦復發 → 縮回去

**為什麼重要？**
打針是病人的痛點：每次都要請假、家屬接送、心理壓力。**減少打針次數 = 大幅改善生活品質與健保成本**。
faricimab 的 ~80% 患者 1 年後可以 Q12W 以上 → 比 aflibercept 顯著減少打針負擔。

## 9. anatomic 與 functional outcome 的「不平行」

很多眼科 trial 會看到：

> 兩個藥的 **CST / fluid（anatomic）**有顯著差異，但 **BCVA（functional）**差不多。

Cheung 2025 也是這樣。為什麼？

- **CST / fluid 是 disease activity 的客觀 readout** → 對藥很敏感
- **BCVA 是 final functional output**，受很多因素影響：
  - Photoreceptor（感光細胞）已死的不會回來
  - Outer retina structure 已破壞的恢復有限
  - 視神經狀況、白內障、其他眼疾
- 短期（12 週）：解 fluid 快不一定立刻反映在 BCVA
- 長期（48 週+）：anatomic control 好的人比較不容易視力惡化

**對你讀 paper 的意義**：
看到「anatomic 改善但 vision 沒改善」**不要急著說藥沒用**——這是 nAMD 領域常態。

## 12. 一頁 cheat sheet（印一份帶在身上）

| 縮寫 | 想到什麼 |
|---|---|
| **nAMD** | 老人黃斑部底下長壞血管漏水；anti-VEGF 救它 |
| **VEGF-A** | 血管增生訊號，anti-VEGF 主標靶 |
| **Ang-2** | 血管不穩訊號，faricimab 第二標靶 |
| **OCT** | 視網膜超音波（光學版） |
| **CST** | 中央視網膜厚度（μm），越厚通常越糟 |
| **IRF / SRF** | 視網膜內 / 下水泡，anti-VEGF 要消的東西 |
| **BCVA** | 視力（ETDRS letters，0–100），越高越好 |
| **ETDRS letters** | 國際標準視力分；85 ≈ 20/20、55 ≈ 20/100 |
| **Q4W / Q8W / Q12W / Q16W** | 每幾週一針 |
| **T&E** | 治療延伸（病情穩拉長間隔） |
| **faricimab (Vabysmo)** | Roche 雙標靶，2022 上市，最長 Q16W |
| **aflibercept (Eylea)** | Regeneron 標準 VEGF blocker，Q8W |
| **TENAYA / LUCERNE** | Roche faricimab 兩個 phase 3 RCT，今天 paper 的源 |
